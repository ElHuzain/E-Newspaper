# E-Newspaper

A Flutter project for the 10 days course provided by ITI for MTI students.

## Task Divisions

### Pages:
- Headlines page
- Favorites page
- News page
- About page

### Model Class:
- News model
- About model (For about page)

### Custom Widgets
- News Widgets
- Header Widgets

### API
- Newsapi

### State Management
- State managed through the 3 main pages
